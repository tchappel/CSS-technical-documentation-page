Find below an explanation for the most common CSS properties that control typography.

font-family

To change the typeface of text on your web page, you can use the font-family property.

Pay attention to the following points:

  • When the name of a typeface consists of more than one word, it's a best practice to enclose the typeface name in quotes;
  • The font specified in the stylesheet must be installed on a user's computer in order for that font to display when a user visits the web page; read more about web-safe fonts;
  • It's a good design decision to limit the number of typefaces used on a web page to 2 or 3.

In the following example we set the font family of all h1 elements to "Garamond", and that of all p elements to "Courier New":

h1 {
font-family: Garamond;
}

p {
font-family: "Courier New";
}
Fallback Font

Fallback fonts are included in case a visitor's web browser does not support the first font.

Usually, more than one fallback font is included, two or three is standard.

h1 {
font-family: Georgia, serif;
}

In the Example above, the font-family property of all h1 elements is set to the value of "Georgia", with "serif" as a fallback font.

Use Generic font options such as:

  • serif
  • sans-serif
  • cursive
  • monospace
font-size

To change the size of text on your web page, you can use the font-size property.

p {
font-size: 18px;
}

In CSS, size can be assigned using relative values or absolute values:

  • Relative values are calculated based on the nearest ancestor element font size
  • absolute values are not affected by the ancestor element font size
  • Both relative and absolute values are inherited by descendant elements
font-weight

In CSS, the font-weight property controls how thick or bold the element shall appear.

The values of font-weight can be defined with numbers ranging from 100 to 900

Keywords values can be set to normal, bold, lighter, bolder or inherit:

  • normal = 400
  • bold = 700
  • lighter/bolder are relative to the inherited font weight
p {
  font-weight: bold;
}
font-style

this property can be used to add and remove italic style or oblique style to an element

there are 3 values:

  • italic
  • oblique
  • normal (removes italic or oblique style)

If the font family does not have a specific italic or oblique typeface, the browser will simulate it by sloping the regular font face glyph.

span {
  font-style: italic;
}
